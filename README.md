### Порядок работы:

1. **Запусти сервер MongoDB** (если у тебя локальный MongoDB):

   ```sh
   mongod
   ```

2. **Запусти сервер**:

   ```sh
   npm run server
   ```

3. **Запусти React-приложение**:
   ```sh
   npm run dev
   ```
